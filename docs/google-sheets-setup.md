# Подключение Google Sheets к RSVP-форме

Этот файл — пошаговая инструкция. Делается один раз, ~15 минут.

---

## 1. Создать таблицу

1. Откройте [sheets.new](https://sheets.new) в своём Google-аккаунте.
2. Назовите её, например, **«RSVP — Артём и Олеся»**.
3. **Лист 1 — переименуйте в «Ответы»** (правой кнопкой по вкладке внизу → Переименовать).
4. В первой строке листа «Ответы» создайте заголовки **по-русски**:

   | A | B | C | D | E | F |
   |---|---|---|---|---|---|
   | Дата ответа | Имя и фамилия | Будет на свадьбе | Гостей с собой | Напитки | Комментарий |

5. Закрепите первую строку: **Вид → Закрепить → 1 строка**.
6. Можно отформатировать: выделить строку 1, **Жирный** (`Ctrl/Cmd+B`), залить светло-серым.

---

## 2. Создать лист «Сводка»

1. Внизу таблицы — кнопка **+** (добавить лист).
2. Переименуйте новый лист в **«Сводка»**.
3. Вставьте такую таблицу:

   | A | B |
   |---|---|
   | **Подтвердили присутствие** | `=COUNTIF(Ответы!C:C; "да")` |
   | **Не смогут прийти** | `=COUNTIF(Ответы!C:C; "нет")` |
   | **Всего гостей (с +N)** | `=COUNTIF(Ответы!C:C; "да") + SUM(Ответы!D:D)` |
   |   |   |
   | **Напитки** |   |
   | Красное вино | `=COUNTIF(Ответы!E:E; "*Красное вино*")` |
   | Белое вино | `=COUNTIF(Ответы!E:E; "*Белое вино*")` |
   | Шампанское | `=COUNTIF(Ответы!E:E; "*Шампанское*")` |
   | Виски / коньяк | `=COUNTIF(Ответы!E:E; "*Виски*")` |
   | Водка | `=COUNTIF(Ответы!E:E; "*Водка*")` |
   | Без алкоголя | `=COUNTIF(Ответы!E:E; "*Не буду*")` |

   > **Важно:** в формулы вставляйте **знак точки с запятой** `;` — у российских Sheets разделитель параметров `;`, а не `,`. Если у вас английская локаль Sheets — замените `;` на `,`.

4. Формулы будут пересчитываться автоматически при каждом новом ответе. На «Сводке» можно зайти в любой момент и сразу увидеть актуальные цифры.

> ❌ **Не ставьте сводку на строке 80 листа «Ответы»** — скрипт использует `appendRow`, который пишет в конец данных. Если на 80-й строке будет сводка, новые ответы упадут после неё (со строки 81), и проверка дублей сломается. Отдельная вкладка — безопаснее.

---

## 3. Создать Apps Script

1. В этой же таблице: **Расширения → Apps Script**.
2. Удалите всё, что там есть, и вставьте код из файла **`google-apps-script.gs`** (см. рядом с этой инструкцией).
3. Нажмите **Сохранить** (значок дискеты или `Ctrl/Cmd+S`).
4. Дайте проекту имя, например, **«rsvp-handler»**.

---

## 4. Развернуть как Web App

1. В Apps Script: **Развернуть → Новое развертывание**.
2. У шестерёнки рядом со словом «Выбрать тип», выбрать **Веб-приложение** (Web app).
3. Заполнить:
   - **Описание:** `rsvp v1`
   - **Запускать как:** *Me (your@gmail.com)* — скрипт работает от вашего имени и пишет в вашу таблицу.
   - **Кто имеет доступ:** *Anyone* — любой человек на свадьбе сможет отправить форму. **Это не значит, что любой увидит таблицу.** Таблица остаётся приватной.
4. Нажать **Развернуть**.
5. Google спросит разрешение — нажмите *Authorize access*, выберите свой аккаунт, **«Дополнительно» → «Перейти к rsvp-handler (небезопасно)»** (так Google помечает свои же скрипты), **Разрешить**.
6. Скопируйте **Web app URL** — он выглядит так:
   ```
   https://script.google.com/macros/s/AKfycb.../exec
   ```

---

## 5. Вставить URL в приглашение

В файле `parts/RSVP.jsx` найти строчку:

```js
const SHEETS_ENDPOINT = ''; // ← вставить сюда URL Web App
```

И заменить `''` на скопированный URL в кавычках:

```js
const SHEETS_ENDPOINT = 'https://script.google.com/macros/s/AKfycb.../exec';
```

Сохранить, задеплоить. Готово.

---

## 6. Проверить

Откройте приглашение, заполните RSVP с тестовым именем «Проверка», отправьте. На листе «Ответы» появится новая строка, на «Сводке» цифры обновятся. Теперь попробуйте отправить ещё раз с тем же именем — должно прийти сообщение «такое имя уже подтверждено».

---

## Обновления Apps Script

Если позже захотите изменить логику скрипта:
1. Откройте Apps Script, отредактируйте код.
2. **Развернуть → Управление развертываниями → ✏️** (карандаш).
3. Выберите версию **Новая версия**, нажмите **Развернуть**.
4. **URL не меняется**, ничего в приглашении трогать не нужно.

---

## Полезные дополнения

- **Email-уведомление при каждом ответе:** в скрипте уже заготовлена константа `NOTIFY_EMAIL` — впишите свой адрес и сохраните.
- **Защита от случайного спама:** в скрипте и в `parts/RSVP.jsx` есть совпадающая константа `SECRET` / `SHEETS_SECRET`. Это shared-секрет: фронт всегда добавляет его в payload, скрипт проверяет. Если значения не совпадают — скрипт отвечает `forbidden` и ничего не пишет. **Важно:** значения должны совпадать в обоих файлах. Если меняете в одном — меняйте и в другом, и не забудьте сделать новое развёртывание Apps Script (Manage Deployments → ✏️ → New version → Deploy).
- **Уведомления в Telegram:** нужен бот через [@BotFather](https://t.me/BotFather). Готов помочь добавить, если решите.
- **Развёрнутая статистика по напиткам с +N:** если хотите, чтобы напитки учитывались с учётом количества гостей (т.е. «+2» считается как 3 человека), напишите — пришлю усовершенствованную формулу.
